Basic Linux command that we will use it everyday 

1] top - Display linux processes.

3] ps - It reports a snapshot of the current processes.

4] grep - It prints line that matches pattern.

5] pkill- It is use to kill process by its name.

6] uptime - It tells you how long the system has been runnig.

7] free - It display amount of free and used memory in the system.

8] pwd - It shows the current working directory.

9] cd - It is used for change directory.

10] ls - It list files and directories in current working directory.

11] mkdir - It is used to create new directory.

12] touch - It is used to create new file.

13] mv - It is used to moves and renames the file.

14] rm - It is used to remove files or directories.

15] cat - It displays file contents.

16] ip a - Displays IP address and network interface.

17] ifconfig - It is used to configure network interface.

18] ping - It sends request to network hosts.

19] curl - It is used to transfer data or urls.

20] dig - It is used to query DNS servers for domain information.
